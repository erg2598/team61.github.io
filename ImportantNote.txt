1. To generate the documentation, you have to use Maven

    In the terminal, run:
    mvn javadoc:javadoc


It throws a bunch of errors but for some reason still generates all of the files just fine. It's set to
put everything inside the docs folder so that the GitHub will host the site correctly.

2. When filling in the documentation, you can type in '/**' and then hit enter and VSCode will autofill everything you need to add comments for.
